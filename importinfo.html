How To Import A Private Key Into A SHND GUI

Step 1:   Make sure you have the latest wallet GUI. You can get the latest wallet GUI from here: Click Here

Step 2:   If this is the first time you're downloading and using a new wallet GUI, then you can speed up the syncing by using the BootStrap. Click Here to get the BootStrap and read the "SHND Instructions text file" that's included in the BootStrap download for how to use the BootStrap.

Step 3:   Your wallet GUI must be fully synced and blocks must be up-to-date. You can check if your blocks are up-to-date at the explorer website. Click Here to go to the explorer website.

Step 4:   While you have your wallet GUI open; click Help, then click Debug window, then click the Console tab.

Step 5:   Now type the info in the textbox. Which is: "importprivkey Your_Private_Key" and should look exactly like the example in "Step 6".

Step 6:   importprivkey 74udndkrid9939dnFej37nBeu488jabcdefGhiJ92

Step 7:   Then press Enter on your keyboard.
